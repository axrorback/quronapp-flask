# 💚 Quran Audio Flask App

A modern Flask web application that lets users search and listen to full Surahs (chapters) of the Holy Qur'an in beautiful audio format. Built with `Flask`, `Bootstrap 5`, and the [alquran.cloud](https://alquran.cloud/api) API.

## 🌐 Live Features

- ۩ Search Surahs by name or number in Uzbek
- 🎵 Play audio of each Surah using high-quality recitations
- 📋 View the full Surah text in Arabic (Uthmani script)
- ⌚ Tashkent time zone based local time display
- ℹ️ About page describing the app

## ⚡ Technologies Used

- Python 3.x
- Flask
- Jinja2 Templates
- Bootstrap 5 & Bootstrap Icons
- CSS3
- [alquran.cloud API](https://alquran.cloud/api)

## 🚀 Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/quran-audio-flask-app.git
cd quran-audio-flask-app
```

2. **Create virtual environment**
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. **Install dependencies**
```bash
pip install -r requirements.txt
```

4. **Run the app**
```bash
flask run
```

5. **Visit in browser**
```
http://localhost:5000
```
